# BillMaker.io - Invoice Management System

A sleek, modern, and professional invoice management system designed for businesses to track clients, services, payments, and business analytics.

## 🚀 Features

- **Responsive Dashboard**: Real-time overview of business performance.
- **Client Management**: Track client details and billing history.
- **Service Catalog**: Manage predefined services and pricing.
- **Invoice Engine**: 
  - Create professional invoices with automatic sequential IDs.
  - Support for GST (18%) calculations.
  - Responsive web view and printable A4 PDF layouts.
- **Payment Tracking**: Record and track payments (Cash, UPI, Bank Transfer, etc.).
- **Smart Analytics**: 
  - Revenue Timeline (Area Charts).
  - Client Revenue Distribution (Pie Charts).
  - Payment Method Breakdown (Progress Trackers).
- **Mobile First**: Fully responsive across all devices.

## 🛠 Tech Stack

- **Frontend**: React.js, Vite, Tailwind CSS, Recharts, Lucide Icons.
- **Backend**: Node.js, Express.js.
- **Database**: MongoDB Atlas.
- **Auth**: JWT (JSON Web Tokens).

## 📦 Installation

### Prerequisites
- Node.js (v14+)
- MongoDB Atlas Account

### Backend Setup
1. Navigate to the backend directory:
   ```bash
   cd backend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Create a `.env` file (see `.env.example`) and add your MongoDB URI and JWT Secret.
4. Start the server:
   ```bash
   npm run dev
   ```

### Frontend Setup
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
